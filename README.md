changbaSpider
==============

1.  程序：唱吧爬虫  
2.  版本：0.2  
3.  作者：zhm  
4.  日期：2016-11-07  
5.  语言：Python 2.7  
6.  操作：输入用户主页编号并按下回车  
7.  说明：用户主页编号是指用户主页链接尾部的一串数字，例如主页编号为33085747
8.  功能：依次下载该用户唱的歌曲    

2016-11-07 update：
--
1、可以下载作品当中的视频
2、可以使用多线程下载用户作品
3、可以设置同时下载的线程数threadnum
